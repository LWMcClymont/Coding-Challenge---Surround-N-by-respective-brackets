# Coding-Challenge---Surround-N-by-respective-brackets

Task:

Given a number N, surround each digit D with D brackets. 

See example below. 

Input format:

First line will contain number N.

Output format:

The result will be displayed.

Example:

input:
- `123978`

Output: 
- `(1(2(3((((((9))7(8))))))))`
